# 💜 Happy Birthday Kartik

A personal birthday webpage built with pure HTML, CSS & JavaScript.

## Files

```
kartik-bday/
├── index.html   ← main page
├── style.css    ← all styles
├── script.js    ← animations & slider logic
└── README.md
```

## How to Deploy on GitHub Pages

1. Create a new repo on GitHub (e.g. `kartik-bday`)
2. Upload all 3 files (`index.html`, `style.css`, `script.js`)
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)` → Save
5. Your site will be live at:
   `https://your-username.github.io/kartik-bday`

## Features

- 3-slide interactive card with smooth transitions
- Floating ambient text animations
- Sparkle effects
- Touch/swipe support for mobile
- Fully responsive
